# Игра «Угадай животное»

Консольная программа, которая задает пользователю вопросы с ответами да/нет и угадывает задуманное животное. Если программа ошиблась, она обучается. Программа просит назвать животное и придумать отличающий вопрос, после чего вставляет новый узел в дерево. Ведется статистика информативности вопросов и глубины партий.

## Возможности

- Угадывание животного через бинарное дерево решений с рекурсивным спуском
- Возврат на шаг назад во время игры (команда «назад»)
- Обучение при неверном угадывании, то есть добавление нового животного и вопроса в дерево
- Подсчет количества животных в базе
- Средняя глубина партий за все время и за последние пять игр
- Рейтинг вопросов по доле успешных угадываний

## Используемые структуры данных и алгоритмы

- Бинарное дерево решений. Каждый внутренний узел хранит вопрос с ветками «да» и «нет», листья хранят названия животных. Обход выполняется рекурсивным спуском по ответам пользователя.
- Стек для истории вопросов. Каждый заданный вопрос с выбранным ответом помещается в стек, что позволяет вернуться на предыдущий шаг командой «назад».
- Сортировка выбором для рейтинга вопросов. Вопросы сортируются по убыванию доли успешных угадываний и выводятся в статистике.
- Префиксные суммы для анализа глубины. Длина каждой партии сохраняется в массиве, из которого строится массив накопленных сумм. Среднее за любой диапазон партий вычисляется за O(1).

## Структура проекта

- `node.py` хранит класс `Node` и функции `animal` и `question`
- `stack.py` хранит класс `Stack`
- `decision_tree.py` хранит класс `DecisionTree` со всей логикой работы с деревом
- `stats.py` хранит функции префиксных сумм `build_prefix` и `average_in_range`
- `game.py` хранит класс `Game`, который собирает партию и выводит статистику
- `main.py` точка входа с меню

## Запуск

После запуска на экран выводится меню, команда выбирается вводом ее номера.

Команда 1 начинает новую партию. Программа задает вопросы и пытается угадать животное. В любой момент можно ввести «назад», чтобы вернуться на предыдущий вопрос.
Команда 2 показывает статистику, а именно число животных в базе, среднюю длину всех партий, среднюю длину последних пяти и рейтинг вопросов по информативности.
Команда 3 завершает работу программы.

## Внутреннее устройство

Класс `Node` хранит один узел дерева, то есть текст вопроса или название животного, ссылки `yes` и `no` на дочерние узлы, счетчики `asked` и `success` для статистики. Метод `is_animal` возвращает `True`, если узел является листом.

Класс `Stack` реализует стек на основе списка Python с методами `push`, `pop`, `top`, вспомогательными `size` и `is_empty`, а также перебором элементов через `__iter__`.

Класс `DecisionTree` хранит корень дерева и реализует рекурсивный спуск `descend`, подсчет животных `count_animals`, сбор всех вопросов `all_questions`, сортировку `sorted_questions` и обучение `learn`.

Функции `build_prefix` и `average_in_range` в модуле `stats` реализуют расчет средней глубины через префиксные суммы.

Класс `Game` хранит дерево и список длин завершенных партий. Метод `ask` обрабатывает ввод пользователя. Метод `play` управляет одной партией, то есть спуском по дереву, финальным угадыванием, обновлением статистики узлов и добавлением нового животного при ошибке. Метод `show_stats` выводит сводную статистику.
