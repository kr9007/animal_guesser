# Игра «Угадай животное» с деревом решений

## Описание

Консольная игра, в которой программа задаёт пользователю вопросы с ответами да/нет и пытается угадать задуманное животное. Если программа ошибается, она обучается: просит назвать животное и придумать отличающий вопрос, вставляя новый узел в дерево. Ведётся статистика информативности вопросов и глубины партий.

## Возможности

- Угадывание животного через бинарное дерево решений с рекурсивным спуском
- Возврат на шаг назад во время игры (команда «назад»)
- Обучение: добавление нового животного и вопроса при неверном угадывании
- Статистика количества животных в базе
- Средняя глубина партий за всё время и за последние пять игр
- Рейтинг вопросов по доле успешных угадываний

## Используемые структуры данных и алгоритмы

- **Бинарное дерево решений.** Основная структура программы: каждый внутренний узел хранит вопрос с ветками «да» и «нет», листья хранят названия животных. Обход выполняется рекурсивным спуском по ответам пользователя.
- **Стек для истории вопросов.** Каждый заданный вопрос с данным ответом помещается в стек, что позволяет вернуться на любой предыдущий шаг командой «назад».
- **Сортировка выбором для рейтинга вопросов.** Вопросы сортируются по убыванию доли успешных угадываний и выводятся в списке статистики.
- **Префиксные суммы для анализа глубины.** Длина каждой партии сохраняется в массиве, по которому строится массив накопленных сумм. Среднее за любой диапазон партий вычисляется за O(1).

## Команды меню

После запуска программа предлагает выбрать действие вводом его номера.

- **1 — начать новую партию.** Программа задаёт вопросы и пытается угадать животное. В любой момент можно ввести «назад», чтобы вернуться на предыдущий вопрос.
- **2 — показать статистику.** Выводится число животных в базе, средняя длина всех партий, средняя длина последних пяти и рейтинг вопросов по информативности.
- **3 — выйти из программы.**

## Внутреннее устройство

Класс `Node` хранит один узел дерева: текст вопроса или названия животного, ссылки `yes` и `no` на дочерние узлы, а также счётчики `asked` и `success` для подсчёта статистики. Метод `is_animal` возвращает `True`, если узел является листом.

Класс `Stack` реализует стек на основе списка Python с методами `push`, `pop`, `top` и вспомогательными `size` и `is_empty`.

Класс `DecisionTree` хранит корень дерева и реализует рекурсивный спуск `descend`, подсчёт животных `count_animals`, сбор всех вопросов `all_questions`, сортировку `sorted_questions` и обучение `learn`.

Глобальный список `games` хранит длины завершённых партий. Функции `build_prefix` и `average_in_range` реализуют расчёт средней глубины через префиксные суммы.

Функция `play` управляет одной партией: спуском по дереву, финальным угадыванием, обновлением статистики узлов и добавлением нового животного при ошибке.

Функция `show_stats` выводит сводную статистику: количество животных, средние длины партий и отсортированный рейтинг вопросов.
